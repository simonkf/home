ICS Definitions

CPU [Central Processing Unit ]
the "brain of the computer", microprocess
directory
a logical collection of files stored under a single name.
file
a file is a logical collection of information stored under a single name.
gif
a lossless format for image files that supports both animated and static images.
information
words, numbers, pictures and sound which have meaning to us.
jpeg
a format for compressing image files.
logical
the opposite of pyhsical.
png
a raster graphics file format that supports lossless data compression.
primary storage
is the temporary storage in the computer.
secondary storage
non volatile, permanent storage.
it is non-volatile storage that is not under the direct control of a computer's central processing unit (CPU) or does not directly interact with an application.
